Official Store Locator Plus Repository

You have found the official code repository for Store Locator Plus. Store Locator Plus is a plugin for WordPress that makes it easy to manage tens or tens-of-thousands of locations while providing a easy-to-use store finder for your site visitors.

The repository hosted here includes the latest DEVELOPMENT as well as production code. The code in the WordPress SVN repository is the public final release copy of the product and does not contain the development tools or various development branches.
